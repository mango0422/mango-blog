# 기본 타입 (Primitive Types)

Java는 총 8가지의 기본 타입을 제공합니다. 이들은 객체가 아닌 실제 값을 저장합니다.

| 그룹   | 타입                           |
| ------ | ------------------------------ |
| 정수형 | `byte`, `short`, `int`, `long` |
| 실수형 | `float`, `double`              |
| 문자형 | `char`                         |
| 논리형 | `boolean`                      |

<Callout type="info" title="Java 17 기준">
  이 섹션에서 다루는 내용은 **Java 17 LTS**를 기준으로 합니다.
</Callout>

## 1. 정수형 타입

| 타입    | 크기 (비트) | 크기 (바이트) | 범위                                                   | 기본값 |
| ------- | ----------- | ------------- | ------------------------------------------------------ | ------ |
| `byte`  | 8           | 1             | -128 ~ 127                                             | 0      |
| `short` | 16          | 2             | -32,768 ~ 32,767                                       | 0      |
| `int`   | 32          | 4             | -2,147,483,648 ~ 2,147,483,647                         | 0      |
| `long`  | 64          | 8             | -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807 | 0L     |

- **리터럴 표기법**:

  - `byte`: 접미사 없음 (예: `100`)
  - `short`: 접미사 없음 (예: `1000`)
  - `int`: 접미사 없음 (예: `100000`)
  - `long`: 접미사 `L` 또는 `l` (예: `100000L`)

- **형변환 규칙**:
  - 작은 타입에서 큰 타입으로 자동 형변환 (예: `byte` → `short` → `int` → `long`)
  - 큰 타입에서 작은 타입으로 명시적 형변환 필요 (예: `long` → `int`)
  - 형변환 시 데이터 손실 가능성 주의

## 2. 실수형 타입

| 타입     | 크기 (비트) | 크기 (바이트) | 범위                                    | 기본값 |
| -------- | ----------- | ------------- | --------------------------------------- | ------ |
| `float`  | 32          | 4             | 약 ±1.4E-45 ~ ±3.4E38 (7자리 정밀도)    | 0.0f   |
| `double` | 64          | 8             | 약 ±4.9E-324 ~ ±1.7E308 (15자리 정밀도) | 0.0d   |

- **리터럴 표기법**:

  - `float`: 접미사 `f` 또는 `F` (예: `3.14f`)
  - `double`: 접미사 `d` 또는 `D` (예: `3.14d`) 또는 접미사 없음 (예: `3.14`)

- **형변환 규칙**:
  - `float`에서 `double`로 자동 형변환
  - `double`에서 `float`로 명시적 형변환 필요
  - 정수형과 실수형 간 형변환 시 주의 (소수점 손실 가능성)

## 3. 문자형 타입

| 타입   | 크기 (비트) | 크기 (바이트) | 범위       | 기본값   |
| ------ | ----------- | ------------- | ---------- | -------- |
| `char` | 16          | 2             | 0 ~ 65,535 | '\u0000' |

- **리터럴 표기법**:

  - 작은따옴표로 감싼 단일 문자 (예: `'A'`, `'\n'`, `'\u0041'`)

- **형변환 규칙**:
  - `char`는 정수형(`int`)과 상호 변환 가능
  - `char`에서 `int`로 자동 형변환
  - `int`에서 `char`로 명시적 형변환 필요 (범위 초과 시 데이터 손실)

## 4. 논리형 타입

| 타입      | 크기 (비트)     | 크기 (바이트)   | 값                  | 기본값  |
| --------- | --------------- | --------------- | ------------------- | ------- |
| `boolean` | JVM에 따라 다름 | JVM에 따라 다름 | `true` 또는 `false` | `false` |

- **리터럴 표기법**:

  - `true` 또는 `false`

- **형변환 규칙**:
  - `boolean`은 다른 기본 타입과 형변환 불가
  - 조건문 및 논리 연산에만 사용

## 요약

Java 17의 기본 타입 8개는 각각 고유한 크기, 범위, 리터럴 표기법, 형변환 규칙을 가지고 있습니다. 정수형과 실수형은 크기에 따라 자동 및 명시적 형변환이 가능하며, 문자형은 정수형과 상호 변환이 가능합니다. 논리형은 다른 타입과 형변환이 불가능하므로 주의가 필요합니다.
각 타입의 특성을 이해하고 적절히 활용하는 것이 중요합니다.
